# GuardianX - Task Manager for Manual Malware Detection

A specialized task manager designed for security analysts and advanced users to identify and analyze suspicious processes through manual inspection.

## Project Structure

```
GuardianX-Source/
├── Models/              # Data models
├── Services/            # Core services (process analysis, network monitoring, etc.)
├── ViewModels/          # MVVM view models
├── Views/               # XAML views and code-behind
├── App.xaml             # Application definition
├── MainWindow.xaml      # Main application window
└── SecureTaskManager.csproj  # Project file
```
